# Android Studio 유용한 단축키



Preferences : `cmd + ,`

Line 들여쓰기 정리 : `cmd + I`

Delete Line : `cmd + D`

Replace : `cmd + F`

불필요 import 정리 : `cmd + shift + O`

Method 보기 : `cmd + O`

최근 편집한 파일 팝업 : `cmd + shift + E`

활성 편집기 탭 닫기 : `cmd + W`

줄 이동 : `cmd + L`

Genereate : `cmd + N`



---

맥북 한/영 키 변경

1. `한/A` 키 클릭
